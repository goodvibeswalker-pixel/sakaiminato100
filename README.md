# 境港市100のこと

境港市の今を知り、未来をつくるための100テーマ型ホームページです。

このサイトは、境港市の魅力や課題を「数字」と「現場」の両方から見える化し、市民・事業者・行政・学校が話し合う入口になることを目指しています。

## GitHub Pagesで公開する場合

1. GitHubで `sakaiminato100` リポジトリを作成
2. このフォルダの内容を `main` ブランチへ push
3. GitHubの `Settings` → `Pages` で公開元を `Deploy from a branch` に設定
4. Branchを `main`、Folderを `/root` に設定

公開URLの例:

```text
https://goodvibeswalker-pixel.github.io/sakaiminato100/
```
